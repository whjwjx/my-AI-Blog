## 🛠️ Building the Blog: Tech Stack and Philosophy

This blog is built on the [Tailwind Nextjs Starter Blog](https://github.com/timlrx/tailwind-nextjs-starter-blog) template, with deep AI integration during the development process.

### 1. Core Tech Stack
- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/) - Exceptional performance and SEO, leveraging React 19 features.
- **Styling**: [Tailwind CSS 4.x](https://tailwindcss.com/) - The first choice for responsive design with modern CSS variables.
- **Content Engine**: [Contentlayer2](https://www.contentlayer.dev/) - Transforms Markdown/MDX into type-safe JSON with fast incremental builds.
- **Enhancements**: [Pliny](https://github.com/timlrx/pliny) - Powers core features like comments (Giscus), analytics, and search (Kbar).
- **Text Processing**: **Remark/Rehype** plugins (e.g., `rehype-katex`, `rehype-prism-plus`) for perfect math rendering and code highlighting.

### 2. Design Evolution: Minimalist Modern
I've customized the template to follow a **"Minimalist Modern"** philosophy, inspired by [farhad.my](https://farhad.my):
- **Content First**: Removed unnecessary borders and decorations to keep focus on the text.
- **Breathability**: Increased whitespace and set line-height to `relaxed` for better reading.
- **Subtle Interactions**: Used `hover:bg-gray-100/50` for clean, dynamic feedback.

### 3. AI-Assisted Development
The UI refactoring and feature enhancements were done using the **Trae IDE**. Pair-programming with AI allowed me to implement complex logic faster and focus more on deep thinking and content creation.
