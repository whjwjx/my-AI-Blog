# Tailwind Nextjs Starter Blog

本项目基于 [timlrx/tailwind-nextjs-starter-blog](https://github.com/timlrx/tailwind-nextjs-starter-blog) 开发。

## 快速启动

### 安装依赖

```bash
yarn
```

如果您使用的是 Windows，可能需要运行：

```bash
$env:PWD = $(Get-Location).Path
```

### 本地开发

运行开发服务器：

```bash
yarn dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

### 生产构建

```bash
yarn build
```
